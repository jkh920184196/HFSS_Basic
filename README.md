# HFSS入門課程

### 課程目的

介紹HFSS操作環境並以簡單範例說明完整的建模、模擬及資料分析流程。讓使用者可以充分了解HFSS功能並將其運用在自己的工作當中。

### 課程地點

ANSYS台北訓練教室
