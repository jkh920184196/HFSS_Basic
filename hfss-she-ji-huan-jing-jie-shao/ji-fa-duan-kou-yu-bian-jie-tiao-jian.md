# 激發與端口

### Excitation(激發)

大致上可以分成以下三類

* Port:可以用來計算網路模型(S, Y, Z)
* Field:近場或遠場或磁偏
* Source:電壓源/電流源

### Port(端口)

|              | Modal | Terminal |
| ------------ | :---: | :------: |
| Wave Port    |   V   |     V    |
| Lumped Port  |   V   |     V    |
| Circuit Port |   V   |     V    |
| FLoquet Port |   V   |          |
